# Projet Twitter MFT 

## Lancer le projet

 1- ````composer install````
 
 2- Mettre vos identifiants dans le fichier .env
 
 3- ``symfony server:start
 ``
 
 4- Créer votre BDD : ``php bin/console d:s:u --force``
 
 3- Load le jeux de données``php bin/console doctrine:fixtures:load
``

## pages

- /login 
- /account
- /feed
- /register


## Fonctionnalitées
